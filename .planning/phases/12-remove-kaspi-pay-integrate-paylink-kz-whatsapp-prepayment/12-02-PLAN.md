# Plan 12-02: Booking route cleanup + WhatsApp prepayment button + service form deposit removal

## Goal
Remove all Kaspi deposit logic from the booking creation route. Replace the `WaitingForPaymentScreen` in `components/booking-form.tsx` with a WhatsApp deep-link button that pre-fills a professional Russian booking confirmation template. Remove the Kaspi deposit configuration from `components/service-form.tsx`.

## Context
After Plan 12-01 removed Kaspi files and Prisma fields, this plan cleans up the consuming code. All bookings should now be created as `CONFIRMED` immediately. The deposit/prepayment UX is replaced with a WhatsApp button that opens a pre-filled message to the business.

## Requirements covered
- PIV-03: Booking route cleanup
- PIV-04: WhatsApp prepayment button
- PIV-05: Service form deposit UI removal

**Step 2f — Add WhatsApp prepayment button to the booking confirmation step.**
After the booking is confirmed (in the success state where the user sees the confirmation), add a WhatsApp CTA section. This appears on the confirmation success view, after the booking details are shown.

The button should only render when `tenantPhone` is a non-empty string.

Build the WhatsApp URL dynamically using booking details:
```ts
function buildWhatsAppPrepaymentUrl(params: {
  tenantPhone: string
  guestName: string
  serviceName: string
  date: string       // formatted date string e.g. "5 апреля 2026"
  time: string       // formatted time e.g. "14:00"
  price: number | null
  currency: string
}): string {
  const priceText = params.price
    ? `${new Intl.NumberFormat('ru-RU').format(params.price)} ${params.currency}`
    : ''
  
  const template = [
    `Здравствуйте! Я хочу подтвердить бронирование.`,
    `📋 Услуга: ${params.serviceName}`,
    `📅 Дата и время: ${params.date}, ${params.time}`,
    priceText ? `💰 Стоимость: ${priceText}` : '',
    ``,
    `Мне необходимо внести предоплату. Подскажите, как это сделать?`,
  ].filter(Boolean).join('\n')

  const phone = params.tenantPhone.replace(/[^0-9]/g, '')
  return `https://wa.me/${phone}?text=${encodeURIComponent(template)}`
}
```
