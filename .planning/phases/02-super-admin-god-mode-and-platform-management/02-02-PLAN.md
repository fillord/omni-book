**Data fetching (server-side):**
```typescript
const PLAN_MRR: Record<string, number> = { FREE: 0, PRO: 29, ENTERPRISE: 99 }

const [planCounts, totalTenants] = await Promise.all([
  basePrisma.tenant.groupBy({
    by: ['plan'],
    where: { planStatus: 'ACTIVE' },
    _count: { _all: true },
  }),
  basePrisma.tenant.count({ where: { planStatus: 'ACTIVE' } }),
])

const mrr = planCounts.reduce((sum, row) => sum + PLAN_MRR[row.plan] * row._count._all, 0)

const chartData = planCounts.map(row => ({
  name: row.plan,
  count: row._count._all,
  mrr: PLAN_MRR[row.plan] * row._count._all,
}))
```

**Layout structure:**
1. Header: `<h1>Аналитика платформы</h1>` + subtitle
2. KPI stat cards row (3 cards, `grid grid-cols-1 sm:grid-cols-3 gap-4`):
   - Card 1: "MRR" — display `$${mrr}` with DollarSign icon
   - Card 2: "Активных компаний" — display `totalTenants` with Users icon
   - Card 3: "Средний MRR/компания" — display `$${totalTenants > 0 ? Math.round(mrr / totalTenants) : 0}` with TrendingUp icon
3. Two-column grid (`grid gap-6 lg:grid-cols-2`):
   - Left: "Распределение по тарифам" — Recharts BarChart showing plan counts (X: plan name, Y: count, bar fill colors: FREE=#94a3b8, PRO=#4299e1, ENTERPRISE=#8b5cf6)
