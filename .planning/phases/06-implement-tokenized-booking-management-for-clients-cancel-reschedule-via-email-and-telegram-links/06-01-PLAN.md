**Step 1: Update Prisma schema**

In `prisma/schema.prisma`, add `manageToken` field to the Booking model. Insert it after the `telegramChatId` field (line 195) and before the `resource` relation:

```prisma
  manageToken       String?       @unique
```

The field is nullable (`String?`) because existing bookings in the database do not have tokens. The `@unique` constraint ensures token lookups are indexed and fast. No default value needed — token is set at creation time.
