**Props:**
```typescript
type ManualBookingSheetProps = {
  open: boolean
  onOpenChange: (open: boolean) => void
  tenantSlug: string
  resources: ResourceOption[]
  services: ServiceOption[]
  onSuccess: () => void  // callback to refresh bookings list
}
```

**Component structure:**

1. **State management:** Use `useState` for form fields (selectedDate, selectedResourceId, selectedServiceId, selectedSlot, clientName, clientPhone) and UI state (slots, slotsLoading, submitting).

2. **Service filtering:** When `selectedResourceId` changes, filter `services` to show only those linked to the selected resource: `services.filter(s => s.resources.some(r => r.resourceId === selectedResourceId))`. Reset `selectedServiceId` and slots when resource changes.

3. **Slot loading:** When `selectedDate`, `selectedResourceId`, AND `selectedServiceId` are all set, fetch from `/api/bookings/slots`:
```typescript
const params = new URLSearchParams({ tenantSlug, date: selectedDate, resourceId: selectedResourceId, serviceId: selectedServiceId })
const res = await fetch(`/api/bookings/slots?${params}`)
const json = await res.json()
setSlots(json.slots ?? [])
```
Use `useEffect` with dependency on the three selection values. Show loading state while fetching.

4. **Form submission:** On submit:
   - Find the selected slot object to get `start` and `end` times
   - Call `createManualBooking({ date: selectedDate, resourceId: selectedResourceId, serviceId: selectedServiceId, startsAt: slot.start, endsAt: slot.end, clientName, clientPhone })`
   - On success: show `toast.success(t('dashboard', 'bookingCreated'))`, call `onSuccess()`, close sheet via `onOpenChange(false)`, reset form
   - On error: show `toast.error(result.error || t('common', 'error'))`

5. **JSX layout** (per UI-SPEC Panel B):
   - `<Sheet open={open} onOpenChange={onOpenChange}>`
   - `<SheetContent side="right" className="sm:max-w-md overflow-y-auto bg-[var(--neu-bg)] backdrop-blur-sm">`
   - `<SheetHeader>` with `<SheetTitle>` using i18n key `manualBookingTitle`
   - Form sections in order: Date (input type="date"), Resource (Select), Service (Select, filtered), Time Slots (grid), Client Name (input), Client Phone (input)
   - Each input/select: `className="neu-inset rounded-xl bg-[var(--neu-bg)]"`
   - Each `<label>` with proper `htmlFor` linking
   - `<SheetFooter>` with submit `<Button className="neu-raised rounded-xl w-full">` using i18n key `saveBooking`

6. **Slot picker grid** (per UI-SPEC):
   - `<div className="grid grid-cols-3 gap-2">`
   - Available slot: `<button className="neu-raised rounded-lg text-sm px-2 py-1 cursor-pointer">`
   - Selected slot: `<button className="neu-inset rounded-lg text-sm px-2 py-1 text-neu-accent">`
   - Each slot button has `aria-label="Select {time}"` for accessibility
   - Empty state: show i18n key `noSlotsAvailable` when slots array is empty and not loading

7. **Accessibility:**
   - `aria-label` on SheetContent
   - `aria-pressed` on slot chips
