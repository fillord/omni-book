    <footer className="bg-zinc-900 text-zinc-400 py-12">

This creates two root nodes which is invalid JSX. Fix by converting the JSX comment to a standard JavaScript comment placed ABOVE the return statement:

Replace lines 15-16 with:

```tsx
  // intentional: fixed dark footer surface — brand design choice, dark in both modes
  return (

```

Do NOT change any className values. Do NOT wrap in a fragment. Do NOT modify any other lines. The only change is moving the comment from inside `return()` as invalid JSX to above `return` as a valid JS comment.
